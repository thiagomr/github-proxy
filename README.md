## Setup

Install dependencies:

```bash
yarn install
```

Setup .env vars following the example:

```
cp .env.sample .env
```

Start app:

```bash
yarn start
```

Run on development:

```bash
yarn start:dev
```

Run on Docker:

```bash
bash docker.sh start
```

Run tests:

```bash
yarn test
```

Run linter:

```bash
yarn lint
```

See API Docs at path: **/api-docs**
